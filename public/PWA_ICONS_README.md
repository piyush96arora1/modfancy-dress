# PWA Icons Required

This app requires PWA icons for installation on mobile devices. You need to create and add the following icon files to the `public` folder:

## Required Icons

1. **icon-192x192.png** - 192x192 pixels (for Android and general use)
2. **icon-512x512.png** - 512x512 pixels (for Android and high-resolution displays)

## Icon Design Guidelines

- Use a square image with your app logo/brand
- Ensure the icon looks good at both sizes
- Use a transparent or solid background that matches your brand
- Keep important elements centered and visible at small sizes
- Consider using a maskable icon format for better Android support

## Quick Generation

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Or create them manually using design tools like Figma, Photoshop, or Canva.
